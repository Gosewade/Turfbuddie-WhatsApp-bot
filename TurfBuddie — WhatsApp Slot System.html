⚙️ How It Works
1
WhatsApp Business API
Connect Twilio / Meta Cloud API to a WhatsApp number. Bot receives messages 24/7.
wa.me/+917000000000
2
Owner Sends Command
Venue owner texts BLOCK/UNBLOCK/SLOTS commands to bot. OTP verified once daily.
BLOCK 10AM
UNBLOCK 5PM
CLOSE ALL
3
Database Updated Instantly
Bot calls TurfBuddie API → Firebase/Supabase slot status updated in <500ms.
PATCH /api/slots/10AM
{ status: "blocked" }
4
Website Auto-Syncs
Real-time listener (Firestore/Supabase) updates the website slot grid instantly. No reload needed.
5
Players Book via WA
Players text BOOK 5PM → bot confirms, creates booking, sends payment link.
BOOK 5PM
→ Confirmed + Pay Link
📲 WA Commands
SLOTS TODAY / SLOTS 30APR
View all slot availability for a date
BOOK [TIME] eg: BOOK 5PM
Player books an available slot
BLOCK [TIME] eg: BLOCK 10AM
Owner blocks a slot (owner only)
UNBLOCK [TIME]
Owner reopens a blocked slot
CLOSE ALL / OPEN ALL
Bulk block/open all slots for a day
REPORT TODAY
Owner gets daily bookings + revenue summary
CANCEL [REF]
Cancel a booking by reference number
